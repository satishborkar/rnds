CSS3 viewport width [vw] and viewport height [vh] unit

Open dialog

CSS 3had intruduced new Unit which will help us to improve coding for width specific deveice.
Viewport Width = vw;
for e.g = 100vw; // this is fullscreen
1 vw = 1/100 of the screen width

Viewport Height = vh;
for e.g = 100vh; // this is fullscreen
1 vh = 1/100 of the screen height
